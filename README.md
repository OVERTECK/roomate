# Roomate

Платформа для поиска соседей по всей России.

**Сайт:** [roomate.ru](https://roomate.ru)

---
/
## О проекте

Веб-сервис, позволяющий людям находить соседей для совместного проживания. Охватывает весь российский рынок. Включает поиск по городам, общение между пользователями, профили и систему рейтингов.

---

## Технический стек

**Frontend:** TypeScript, React, CSS  
**Backend:** C#, ASP.NET Core  
**DevOps:** Docker, Docker Compose, GitHub Actions

---

## Развертывание

```bash
docker-compose up -d
```

Проект полностью контейнеризирован и готов к production.
