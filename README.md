Roomate (сервис поиска соседей для совместного проживания)

Полноценное веб-приложение, развёрнутое в продакшене.
Реализовал интерфейс с публикацией объявлений, фильтрацией по параметрам, личными сообщениями в реальном времени (SignalR/WebSocket) и загрузкой фотографий.
Работал с REST API, выстраивал взаимодействие клиента с бэкендом - аутентификация, авторизация, передача данных.
